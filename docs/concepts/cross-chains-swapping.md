title: Cross chain token swapping
---

Cross chain token swapping is not available for current testnet. This feature will be deployed in Tesnet Phase 2. In Testnet Phase 1, we want to test the correctness and robustness of our Consensus and EVM engines.

Please check our [timeline](/docs/more-info/timeline) for Sisu network's progress. We will give you full update and documentation when Testnet Phase 2 is launched.
